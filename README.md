# Сайт-визитка консультанта по недвижимости

Статический лендинг для персонального бренда консультанта по недвижимости. Включает:

- первый экран с позиционированием и контактным действием;
- блоки услуг, кейсов и процесса работы;
- отзыв клиента и контактный блок;
- адаптивную верстку для мобильных устройств.

## Запуск

Откройте `index.html` в браузере или запустите любой статический сервер в корне проекта.
